# InsertCoin

Application React/Vite pour gérer une collection retrogaming, prête à déployer sur Coolify.

Repo : https://github.com/nickdesi/InsertCoin.git

## Développement

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Coolify

- Type : Dockerfile recommandé
- Port exposé : `80`

Les données restent stockées dans le navigateur via `localStorage`.
